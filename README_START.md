# ELN App local start

Double click `start_eln_app.bat` to start the local ELN service and open:

`http://127.0.0.1:8550/`

The iPhone should use the LAN address shown in Settings inside the app, usually:

`http://<your-windows-ip>:8550/`

Data is stored outside this project folder under:

`C:\Users\Ingester\ELN_Data`
